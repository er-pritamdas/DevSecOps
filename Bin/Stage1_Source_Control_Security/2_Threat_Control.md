| **Weak Authentication**                     | Password-only authentication increases risk of credential theft.                                                    | MFA, SSO, password policies, hardware security keys.                   | [YubiKey](https://www.yubico.com/?utm_source=chatgpt.com), Okta, Microsoft Entra ID                                                                                                                                                                                                                   |

### If you are designing **Source Control Security** for an enterprise, the controls I'd consider mandatory are:

| Priority | Control                   |
| -------- | ------------------------- |
| Critical | MFA                       |
| Critical | Branch Protection         |
| Critical | Pull Request Reviews      |
| Critical | Secret Scanning           |
| Critical | Least Privilege Access    |
| High     | Commit Signing            |
| High     | CODEOWNERS                |
| High     | Audit Logging             |
| Medium   | SSO Integration           |
| Medium   | Repository Access Reviews |

These are the controls you'll most commonly encounter in real DevSecOps environments before the code even reaches the CI/CD pipeline.
